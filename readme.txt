Where-first data science. A dark-first WordPress block theme for Siege Analytics.

== Description ==

A modern WordPress block theme (Full Site Editing) for siegeanalytics.com. Dark-first design with Inter typography, JetBrains Mono data readouts, and Signal Green accent color.

Features:
* Full Site Editing support — header, footer, nav, colors, and layout all editable from the admin UI
* Blog support with RSS feeds, categories, tags, and archive pages
* Responsive design tested across mobile, tablet, and desktop
* Custom 404 page
* Random coordinate system display in footer (WGS84, UTM, MGRS, State Plane)
* Cross-browser tested on Chrome, Firefox, Safari, and Edge
